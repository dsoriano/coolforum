CoolForum
=========

Coolforum est un forum de discussion développé en PHP et MySQL. Pour en savoir plus, merci de vous rendre sur http://www.coolforum.net.

Après 9 ans sans mises à jour, j'ai décidé dans un premier temps de nettoyer "un peu" le code afin de le rendre un peu plus compatible avec les versions plus récentes de PHP, dans un second temps de placer le code sur Github, en espérant que cela me motive à reprendre le code du forum et à l'améliorer :)

----------

Merci de lire le fichier **!Lisez-moi.txt** pour la procédure d'installation / de mise à jour.
